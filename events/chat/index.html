پیام‌های چت

Chat Message​

پارامترنوعتوضیحات
idstringشناسه‌ی پیام
conversationConversationمکالمه‌ای که پیام در آن ارسال شده است
senderMessageSenderارسال‌کننده‌ی پیام
typestringنوع پیام: TEXT، IMAGE، FILE، VOICE، LOCATION، VIDEO، CONTACT
sent_atTimestamp (int64)زمان ارسال پیام
textstringمتن پیام (در صورتی که نوع پیام TEXT باشد)
image_dataImageDataاطلاعات تصویر (در صورتی که نوع پیام IMAGE باشد)
file_dataFileDataاطلاعات فایل (در صورتی که نوع پیام FILE باشد)
voice_dataVoiceDataاطلاعات صوتی (در صورتی که نوع پیام VOICE باشد)
location_dataLocationDataموقعیت مکان (در صورتی که نوع پیام LOCATION باشد)
video_dataVideoDataاطلاعات ویدئو (در صورتی که نوع پیام VIDEO باشد)
contact_dataContactDataاطلاعات تماس (در صورتی که نوع پیام CONTACT باشد)

Text​

پارامترنوعتوضیحات
textstringمتن پیام (در صورتی که نوع پیام TEXT باشد)

نمونه پیام متنی (Text)​

{
  "id": "e7a08a2c-8ee4-11ef-af85-4e9e9bfa3c12",
  "conversation": {
    "id": "edbb9499-cccc-4d35-b21f-621fa95f2b3e",
    "type": "POST",
    "post_token": "gZd7omnS"
  },
  "sender": {
    "side": "SELLER",
    "type": "BOT"
  },
  "type": "TEXT",
  "sent_at": 1729429926140000,
  "text": "hello world"
}

Conversation​

پارامترنوعتوضیحات
idstringشناسه‌ی مکالمه
typestringنوع مکالمه: POST، BOT
